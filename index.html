This was the final project I created for my Diploma, it was written in the Django framework for python and utilised more SQL, SQLite for the development version and PostgreSQL for the deployment version. This introduced me to Models and inter-relational databases, allowing me to see how these objects work with each other on the back-end. It is a fully functional E-Commerce application also hosted on Heroku, whilst also utilising AWS S3 Buckets to host my static files elsewhere.
